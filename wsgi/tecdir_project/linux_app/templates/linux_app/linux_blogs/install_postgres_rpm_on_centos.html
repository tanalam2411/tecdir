Install PostgreSQL RPM(binary) on Centos

Edit /etc/yum.repos.d/CentOS-Base.repo file and append exclude=postgresql*

$ vi /etc/yum.repos.d/CentOS-Base.repo

[updates]

exclude=postgresql*

[base]

exclude=postgresql*

Get the specific version of RPM link from here - https://yum.postgresql.org/9.6/redhat/rhel-6.8-x86_64/

$ yum install https://yum.postgresql.org/9.6/redhat/rhel-6.8-x86_64/pgdg-centos96-9.6-3.noarch.rpm

$ yum list postgresql*

$ yum install postgresql96-server -y

Default data directory is /var/lib/pgsql/9.6/data

Initialize the database in PGDATA(/var/lib/pgsql/9.6/data)

$ service postgresql-9.6 initdb

Initializing database:

To start PostgreSQL automatically when the OS starts:

$ chkconfig postgresql-9.6 on

Check the postgres service status($ service postgresql-9.6 status) and if it is stopped, start the service using command:

$ service postgresql-9.6 start

Switch to user "postgres" using command $ su - postgres for root user or $ sudo -i -u postgres for sudo user.

$ su - postgres

$ psql

postgres=#

For extensions like dblink etc install postgresql-contrib

Get the RPM link from here - https://yum.postgresql.org/9.6/redhat/rhel-6.8-x86_64/

$ yum install https://yum.postgresql.org/9.6/redhat/rhel-6.8-x86_64/postgresql96-contrib-9.6.2-2PGDG.rhel6.x86_64.rpm
