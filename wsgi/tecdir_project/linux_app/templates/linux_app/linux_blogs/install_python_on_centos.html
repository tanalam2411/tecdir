Install Python on Centos

Default python version

$ vi python -V

Python 2.6.6

Run the following commands before installing python.

$ yum update -y

$ yum groupinstall 'Development Tools' -y

$ yum install openssl-devel bzip2-devel zlib-devel libpcap-devel ncurses-devel sqlite-devel readline-devel tk-devel gdbm-devel -y

Visit https://www.python.org/downloads/ look for the specific Python version you want to install, click on Download then you will be redirected to a new page with multiple package option. Right click on Gzipped tarball and click on copy link address.

Download Python tarball by running the command $ curl -O link you copied

$ pwd

/home/user/downloads

$ curl -O https://www.python.org/ftp/python/2.7.6/Python-2.7.6.tgz

$ tar -zxvf Python-2.7.6.tgz

Add absolute path to --prefix where you would like to install python(where python will get compiled and installed).

$ ./configure --prefix='/home/user/python2.7'

$ make

$ make test

$ make altinstall

Installation completed.

Test newly installed python.

$ /home/user/python2.7/bin/python2.7 -V

Python 2.7.6

Append bin path to PATH env variable to make it available everywhere.

$ export PATH=$PATH:/home/user/python2.7/bin

$ python2.7

Python 2.7.6

>>> exit()

Installing pip

$ curl -O https://bootstrap.pypa.io/get-pip.py

$ python2.7 get-pip.py

$ which pip

/home/user/python2.7/bin/pip

Installing Virtual Environment

$ pip install virtualenv

$ which virtualenv

/home/user/python2.7/bin/virtualenv

$ mkdir workspace && cd workspace

$ virtualenv venv

$ source venv/bin/activate

(venv) $ which python

/home/user/workspace/venv/bin/python

(venv) $ python -V

Python 2.7.6
